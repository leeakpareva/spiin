# Audio Files

This folder contains audio files for the SPIIN music platform.

## File Structure
- Format: MP3 (recommended for web compatibility)
- Quality: 320kbps or higher for premium experience
- Naming convention: `[artist]-[song-title].mp3`

## For Production
When deploying to Vercel:
1. Ensure audio files are properly compressed
2. Consider using a CDN for large media files
3. Add proper CORS headers if needed
4. Test playback across different browsers and devices
